# Nika font project

Nika is a  Persian/Arabic typeface, suitable for text.

# وضعیت توسعه

این فونت در مراحل اولیه طراحی و پیاده سازی قرار دارد و هنوز به حالت پایدار نرسیده است.
در صورت مشاهده هرگونه اشکال لطفا آن را گزارش نمایید.
